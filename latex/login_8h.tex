\doxysubsection{Detailed Description}
Defines the \doxylink{class_login}{Login} class. 

The \doxylink{class_login}{Login} window allows users to enter credentials before accessing the application. If the password matches an administrator password, the \doxylink{class_admin}{Admin} panel is opened. Otherwise the regular \doxylink{class_main_window}{Main\+Window} opens.
